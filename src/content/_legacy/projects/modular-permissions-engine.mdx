## Why this project

Permissions looked simple until product workflows became conditional and role-only logic started to break.

## What changed

We introduced a policy layer with explicit inputs and outputs, then migrated endpoints in slices. UI checks consumed the same policy contract used by APIs.

## Result

Access behavior became consistent and explainable, which improved trust for both users and internal teams.
