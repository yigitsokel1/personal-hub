## Context

The platform had grown into asynchronous processing quickly. Delivery speed improved, but observability lagged behind and incident resolution depended on intuition.

## Approach

We standardized event envelope metadata and enforced propagation rules in shared libraries. Each service emitted lag and retry metrics with the same labels so alerts became comparable across domains.

## Outcome

With replay and correlation in place, on-call engineers could answer "what happened?" with evidence, not guesswork.
