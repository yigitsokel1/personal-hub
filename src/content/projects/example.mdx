# Personal Hub Platform

This sample **project** entry represents a productized system: something you ship, maintain, and iterate on—not a one-off demo.

## What it demonstrates

- Strong separation between **content** and **UI**
- Typed, domain-specific content (projects, work, writing, labs)
- Room to grow without rewriting the core model

## Technical notes

The stack is intentionally boring where it matters: stable runtime, clear boundaries, and schemas you can evolve.

```text
content/     → source of truth
src/app/     → routing and composition
src/lib/     → parsing, validation, helpers
```
